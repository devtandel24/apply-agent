# GenAI Application Engineer — Example Ventures

**Location:** Remote (US)
**Salary:** $115,000 – $155,000
**Posted:** 2026-06-12

## About Example Ventures
Example Ventures is an early-stage startup building AI-native enterprise software.
Our flagship product is an agentic workflow platform that lets operations teams
automate complex, multi-step business processes using large language models.

## Role Overview
As a GenAI Application Engineer you will be at the center of our product development,
designing and building LLM-powered features, agentic pipelines, and developer
integrations. You'll work directly with the founding team in a fast-moving,
high-ownership environment.

## Responsibilities
- Design and implement LLM-powered agents and multi-step agentic workflows using
  Claude, GPT-4, or open-source models
- Integrate with external APIs, webhooks, and enterprise SaaS tools to enable
  real-world automation
- Build RAG systems (chunking, embedding, vector retrieval, re-ranking) for
  grounding agent decisions in customer data
- Write clean, tested Python (FastAPI back-end) and React/TypeScript (front-end)
- Evaluate model outputs, build automated evaluation pipelines, and iterate on prompts

## Requirements
- 2+ years of software engineering or AI/ML engineering experience
- Hands-on experience building with LLM APIs (Anthropic Claude, OpenAI, or similar)
- Solid Python back-end skills (FastAPI, async patterns, REST API design)
- Understanding of RAG architecture, vector databases, and retrieval strategies
- Ability to prototype quickly and ship production-quality code
- Strong communication skills for a fully remote team

## Nice to Have
- Experience with LangChain, LangGraph, CrewAI, or similar agentic frameworks
- Full-stack experience (React, TypeScript)
- Prior startup or early-stage company experience

## What We Offer
- Fully remote, async-first culture
- Competitive salary with meaningful early-stage equity
- $3,000 annual learning and conference budget
- Health benefits from day one
