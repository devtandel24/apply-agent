# AI/ML Engineer — Demo Corp

**Location:** San Francisco, CA (Remote OK)
**Salary:** $110,000 – $150,000
**Posted:** 2026-06-10

## About Demo Corp
Demo Corp builds next-generation analytics products for enterprise customers. Our
ML platform processes billions of events per month, powering real-time recommendations
and anomaly detection across Fortune 500 clients.

## Role Overview
We're looking for an AI/ML Engineer to join our Core ML team. You will design,
train, and deploy machine learning models that power our recommendation and
personalization features. You'll work closely with data engineers and product
managers to deliver measurable improvements in user engagement.

## Responsibilities
- Design and implement ML pipelines from feature engineering through model deployment
- Build and maintain model training infrastructure on AWS (SageMaker, S3, Lambda)
- Develop LLM-powered features using OpenAI and/or Anthropic APIs
- Instrument model monitoring, A/B testing, and automated retraining workflows
- Collaborate with data engineers on feature store design and data quality

## Requirements
- 2+ years of hands-on ML engineering or data science experience
- Strong Python skills; experience with PyTorch or TensorFlow
- Familiarity with AWS ML services (SageMaker, Lambda, S3) or equivalent cloud
- Experience with LLM APIs (OpenAI, Anthropic, Cohere, etc.)
- Solid understanding of SQL and data modeling fundamentals
- BS/MS in Computer Science, Data Science, or related field

## Nice to Have
- Experience with MLflow, Kubeflow, or similar MLOps tooling
- Knowledge of LangChain, LangGraph, or RAG system design
- Contributions to open-source ML projects
